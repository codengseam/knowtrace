# 00 · 项目定位与阿里生态对接

## 一、赛道与出题方

- **赛道**：方向二 · 乡村课堂 AI 助教
- **出题方**：阿里巴巴少年云助学计划（ATH 事业群 + 阿里云百炼）
- **推荐模型**：Qwen3-Max / Qwen3-Omni / Qwen3 端侧小尺寸版 / FunASR
- **奖励**：优秀入围作品获 2-3 所乡村学校赛后合作试点机会 + 乡村骨干教师方案评审与早期试用反馈
- **时间线**：2026-07-16 启动，2026-08-13 初赛材料提交（28 天冲刺）

## 二、痛点背景

### 学生侧：知识断层的恶性循环

- 乡村留守儿童 **52.5% 存在学业辅导困境**，96% 由祖辈照顾，课后作业无人辅导。
- 知识漏洞有**连锁性**：一元二次方程不会 → 一元一次方程基础不牢 → 有理数运算薄弱，层层塌方。
- 现有搜题工具只解决"这道题怎么做"，解决不了"我到底哪里没学会"。

### 教师侧：学情看不清，讲课无的放矢

- 乡村教师普遍**一人多科、跨年级代课**，60% 以上学校存在非专业教师兼课现象。
- 没时间逐本批改错题本，不知道班级整体薄弱点在哪，上课只能"满堂灌"。
- 想针对性出题，但手头没有题库资源，也没时间组卷。

## 三、阿里生态背景（为什么这个比赛做有优势）

### 现有基础设施已就位

- 少年云助学计划已在全国建 **300+ 间 AI 云教室**，2026 年将超 400 间，覆盖 25 省份 30 万师生。
- 基于**阿里云无影云电脑**，算力在云端，终端只要一根网线一块屏幕，天然适配乡村弱硬件。
- 夸克 + 钉钉已联合推出 AI 作业批改 + 班级错题本功能，有十亿级题库打底。

### 生态结合点（差异化机会）

| 现有阿里产品 | 做到了什么 | 缺什么（我们的机会） |
|---|---|---|
| 夸克错题本 | 收录 + 相似题推荐 | 深度知识链路溯源 |
| 钉钉作业批改 | 班级错题汇总 | 薄弱知识点追溯 + 专项补漏方案 |
| 少年云无影云 | 300 间教室基础设施 | 预装的 AI 认知诊断应用 |

我们要做的：在阿里现有能力之上，加一层**知识图谱智能分析层**，从"搜题工具"升级为"诊断系统"。

## 四、项目定位（一句话）

> **夸克负责"搜题"，我们负责"诊断"；钉钉负责"批改"，我们负责"开药方"。**

不是和阿里现有产品竞争，而是做阿里教育生态里缺失的那一层 **"知识诊断智能"**——从单个错题的解答，上升到整个知识体系的漏洞排查和补漏路径规划。

### 三层价值

| 对象 | 从 → 到 |
|---|---|
| 学生 | 从"改一道错题"到"补一片知识" |
| 老师 | 从"凭经验讲课"到"看数据讲课" |
| 乡村场景 | 从"有网才能用"到"弱网也能用" |

## 五、生态对接策略（可对接但不依赖）

### 理想情况（深度打通）

- 直接调用夸克错题本数据，学生在夸克里搜题自动同步到本系统。
- 钉钉班级群直接推送学情报告。

### 现实情况（比赛阶段）

- 夸克没有公开的错题数据开放 API，比赛期间大概率拿不到官方数据权限。
- **应对策略**：
  1. 方案上设计成"可对接夸克"的架构（`ProblemInputAdapter` 抽象接口），展示生态协同思路。
  2. MVP 阶段自己做拍照录题（Qwen3-Omni 完全能搞定），作为兜底方案。
  3. PPT 里重点讲"接入夸克生态后的放大效应"，体现对阿里体系的思考。

### 加分项

强调**少年云无影云电脑**的适配——系统直接部署在少年云云桌面，成为 AI 云教室里的预装应用，这是出题方最想看到的落地路径。

## 六、比赛胜出核心价值点

| 价值维度 | 为什么评委喜欢 |
|---|---|
| **生态协同** | 站在阿里已有产品基础上做增量，不是重复造轮子 |
| **技术深度** | 知识图谱 + 溯源推理有技术壁垒，不是简单套大模型壳 |
| **落地性强** | 直接对接少年云现有 300 间教室试点资源，赛后即落地 |
| **社会价值** | 精准解决乡村教育"知识断层"结构性问题，不是锦上添花 |
| **公益温度** | FunASR 方言语音周报，让不识字祖辈也能听懂孩子学情 |

## 七、技术栈与阿里能力绑定

| 阿里能力 | 在本项目中的具体应用 |
|---|---|
| Qwen3-Omni | 多模态识别学生手写错题及草稿纸 |
| Qwen3-Max | 错因分析、认知路径推理、教研讲义生成 |
| 阿里云百炼 RAG | 真题向量库检索（5000 道初中数学题） |
| 阿里云百炼 Agent | 编排"识题→诊断→检索→排版"工作流 |
| FunASR | 学情周报语音合成（方言/普通话） |
| 无影云电脑 | SaaS 预装部署载体 |

详细集成方案见 [04-百炼平台技术集成](04-百炼平台技术集成.md)。
